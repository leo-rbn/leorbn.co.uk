> Securing a Network Using A Raspberry Pi: A Practical Cybersecurity Project (EPQ)

For my EPQ (Extended Project Qualification), I explored the use of Ad-Blockers and VPNS. To do this i put Headscale-Admin and Pi-Hole on a Raspberry Pi.

For this I had to report on my process of creating this artefact and an evaluation of its effectiveness using both qualitative and quantitative data. This will be linked when the grade for the project is finalised.

SKILLS GAINED: SYSTEM ADMINISTATION, NETWORKING, PROBLEM SOLVING, BASH, RESEARCH, REPORT WRITING

>>> Building a Home Server

I used an old Dell Optiplex to create a home server to experiment with different softwares and services. For flexibility I used Proxmox, this allowed me to easily try different services and run multiple at once.

So far i have tried Jellyfin as a media server and OpenMediaVault as a NAS (Network Attached Storage). I intend to try many more services to experiemnt and see what works for my workflow
